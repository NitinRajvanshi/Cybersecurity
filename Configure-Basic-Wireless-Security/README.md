# Configure Basic Wireless Security Using WPA2

## Overview

This Cisco Packet Tracer lab demonstrates the implementation of WPA2 Personal security on a wireless network.

## Objective

Configure a wireless router with WPA2 Personal security and verify secure connectivity from a wireless client.

## Tools Used

- Cisco Packet Tracer
- Wireless Router
- Laptop Client

## Configuration

### Wireless Settings

SSID: Academy

Security Mode: WPA2 Personal

Encryption: AES

Passphrase: Network123

## Verification Process

### Step 1

Verified initial connectivity by accessing:

www.cisco.pka

### Step 2

Configured WPA2 Personal security on the wireless router.

### Step 3

Detected wireless network:

Academy

### Step 4

Connected using WPA2 authentication and entered:

Network123

### Step 5

Verified successful secure connectivity by accessing:

www.cisco.pka

## Security Concepts Learned

- Wireless Network Security
- WPA2 Personal Authentication
- AES Encryption
- Pre-Shared Key (PSK)
- Secure Wireless Access

## Skills Demonstrated

- Wireless Router Configuration
- Network Authentication
- Wireless Client Setup
- Connectivity Verification
- Basic Network Security
